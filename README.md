## To Run:

* Clone repo to your machine - git clone https://github.com/lroliphant/form-prototype
* cd into directory
* `npm install`
* `npm start`

**To run tests:**
* `karma start`

**To bundle files:**
* `npm run build`

**To watch:**
* `npm run watch`
